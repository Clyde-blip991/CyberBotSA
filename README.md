CyberBotSA - Cybersecurity Awareness Chatbot

Description
A C# console application that educates South African citizens on cybersecurity awareness topics including phishing, password safety, safe browsing, malware, and more.

 Project Structure
- AudioPlayer.cs - Handles voice greeting
- DisplayHelper.cs - Displays ASCII logo
- UIHelper.cs - Console UI formatting
- Chatbot.cs - User interaction loop
- ResponseEngine.cs - Cybersecurity responses
- Program.cs - Entry point

How to Run
1. Open CyberBotSA in Visual Studio 2022
2. Press F5 to run
3. Follow the on-screen prompts
